앱 설치 방법

1. Android

  1. Android 기기에서 Chrome 을 엽니다.
  2. 클래스보드 홈페이지로 이동합니다.
  3. '앱 설치하기'를 탭합니다.
  4. 화면에 표시되는 안내를 따릅니다.

2. IPhone/Ipad

  1. IPhone/IPad 기기에서 Safari 을 엽니다.
  2. 클래스보드 홈페이지로 이동합니다.
  3. '앱 설치하기'를 탭합니다.
  4. 화면에 표시되는 안내를 따릅니다.

크롬, 사파리 이외의 브라우저에서는 설치할 수 없습니다
